# Image Prompts — Mula Platform

**Mission 004 — Premium Graphics System**
**Used with:** Midjourney / DALL-E / Stable Diffusion

## Universal Base Prompt

```
Premium dark-tech enterprise visual for Mula Group, abstract high-end technology ecosystem, cinematic lighting, deep black background, subtle grid, glowing data structures, elegant 3D depth, clean composition, no text, no logos, no people, professional European technology brand, responsive web illustration, ultra high resolution, premium SaaS aesthetic.
```

## Universal Negative Prompt

```
no text, no words, no letters, no logo, no watermark, no people, no stock photo, no cheap 3D icons, no cartoon, no gaming style, no crypto coin style, no clutter, no low resolution, no excessive neon, no generic laptop scene
```

## Prompt Format

```
[BASE PROMPT]
Main concept: [what the image represents]
Color accent: [hex color]
Visual symbols: [specific elements to include]
Mood: [atmospheric description]
Composition: [layout guidance]
Constraints: [what NOT to include]
```

## Usage

1. Copy the universal base prompt and universal negative prompt
2. Add the per-competency specification
3. Generate at 1600x1200 for the prototype
4. Review and iterate
5. Once approved, generate all formats (desktop 3840x2160, card 1600x1200, mobile 1080x1350, OG 1200x630)
