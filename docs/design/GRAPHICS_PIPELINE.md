# GRAPHICS PIPELINE — Mula Group

## Overview
Each of the 7 competencies receives dedicated graphics for use across the platform.

## Design Rules
- No text on images — all text in HTML (SEO, accessibility, responsiveness)
- Dark-tech aesthetic matching Mula Group brand
- Premium quality, consistent style
- Responsive formats (WebP)

## Image Specifications

### Desktop Hero
- 3840×2160px, WebP
- Usage: Full-width hero backgrounds

### Card / Section
- 1600×1200px, WebP  
- Usage: Competency cards, section headers

### Mobile
- 1080×1350px, WebP
- Usage: Mobile hero, mobile cards

### Open Graph / Social
- 1200×630px, WebP
- Usage: og:image meta tag

## File Naming Convention
/public/images/competencies/{competency}-{format}.webp

Examples:
- ai-desktop.webp, ai-card.webp, ai-mobile.webp, ai-og.webp
- digital-desktop.webp, digital-card.webp, digital-mobile.webp, digital-og.webp
- ecommerce-desktop.webp, ecommerce-card.webp, ecommerce-mobile.webp, ecommerce-og.webp
- marketing-desktop.webp, marketing-card.webp, marketing-mobile.webp, marketing-og.webp
- cyber-desktop.webp, cyber-card.webp, cyber-mobile.webp, cyber-og.webp
- construction-desktop.webp, construction-card.webp, construction-mobile.webp, construction-og.webp
- innovation-desktop.webp, innovation-card.webp, innovation-mobile.webp, innovation-og.webp
