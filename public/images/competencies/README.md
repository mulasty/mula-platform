# Competency Images

Premium graphics system for Mula Group's 7 competencies.

## Usage

Images are served via `https://mulagroup.eu/images/competencies/{slug}-{format}.webp`

## Formats

| Format | Dimensions | Use |
|--------|-----------|-----|
| Desktop/Hero | 3840x2160 | Full-width hero backgrounds |
| Card/Section | 1600x1200 | Section backgrounds, cards |
| Mobile | 1080x1350 | Mobile-optimized hero |
| Open Graph | 1200x630 | Social media sharing |

## Rules

- No text on images — text always in HTML
- WebP format, compressed without visible quality loss
- Premium dark-tech aesthetic, consistent across all 7 competencies
- AI-generated style prototype first, then scale
